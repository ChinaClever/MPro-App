全局样式

重置默认样式，尽可能保证在所有浏览器中的一致性。

样式统一

ARM UI 使用 Neat.css 来统一样式，并对部分细节做了调整：
  • 移除了 IE6/7 的 Hack 代码；
  • 移除了已被 W3C 废弃的标签，如 hgroup；
  • html 添加 -webkit-font-smoothing: antialiased; 抗锯齿。

字体定义

字体选用上，ARM UI 优先使用现代字体，注重平台兼容性好、小字显示清晰和阅读辨识度高。

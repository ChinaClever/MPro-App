消息提示

消息提示用于即时反馈用户主动操作后的响应信息。

基本类型

共提供了 5 种不同色彩的消息提示框用于不同场合：

实例
描述

设备列表已更新。

<div class="msg msg-normal">
    <i class="iconfont icon-info"></i>
    <p>设备列表已更新。</p>
</div>

适用于普通类型的消息，起到辅助说明的作用。

设置保存成功！

<div class="msg msg-success">
    <i class="iconfont icon-success_fill"></i>
    <p>设置保存成功！</p>
</div>

适用于成功类型的消息，常用于提示操作已成功完成。

您关闭了提醒功能，将无法收到最新动态。

<div class="msg msg-warning">
    <i class="iconfont icon-prompt_fill"></i>
    <p>您关闭了提醒功能，将无法收到最新动态。</p>
</div>

适用于警告类型的消息，常用于强调某些需要注意的内容。

与服务器连接中断，请检查网络！

<div class="msg msg-error">
    <i class="iconfont icon-warning_fill"></i>
    <p>与服务器连接中断，请检查网络！</p>
</div>

适用于失败类型的消息，一般用于反馈错误、紧急、故障等信息，提醒用户。

使用方法

1. 静态调用

静态调用就是直接在页面上写好消息提示框的 HTML 结构和内容，使其直接显示出来，具体请参考本文“基本类型”里的实例。

2. 动态调用

动态调用是编写 JavaScript 脚本来动态控制消息提示框的提示类型、提示文案、图标、回调、事件等项目。

实例
代码

请点击下面按钮来弹出不同类型的消息提示：

// “普通”按钮点击
$('#demo_eject').msg({
    type: 'normal',  // 消息类型
    text: '这是一条普通消息。'
});

// “成功”按钮点击
$('#demo_eject').msg({
    type: 'success',  // 消息类型
    text: '恭喜你，这是一条成功消息！'
});

// “警告”按钮点击
$('#demo_eject').msg({
    type: 'warning',  // 消息类型
    text: '注意了，这是一条警告消息哦'
});

// “错误”按钮点击
$('#demo_eject').msg({
    type: 'error',  // 消息类型
    text: '出错了，这是一条警告消息！'
});

上面的实例中，弹出的消息提示框会默认在 3s 后自动消失，这个时间可通过 delay 参数来自定义（单位为 ms），当为 0 时，不会自动消失。

// 弹出 5s 后消失
$('#demo_delay').msg({
    delay: 5000,  // 消息停留时间
    text: '该消息提示框将在 5s 后消失'
});

// 弹出后不会自动消失
$('#demo_delay').msg({
    delay: 0,  // 消息停留时间
    text: '该消息提示框不会自动消失'
});

// 手动关闭
$('#demo_delay').msg('close');

动态调用还支持自定义消息提示的小图标，通过设置 icon 参数为图标名称（图标名称参见 组件 → 图标），即可显示相应图标的消息提示框。

$('#demo_icon').msg({
    delay: 0,
    type: 'warning',
    icon: 'icon-feedback_fill',  // 图标名称
    text: '你确定要这么做吗？后果自负哦'
});

$('#demo_icon').msg({
    delay: 0,
    type: 'success',
    icon: 'icon-mail_fill',  // 图标名称
    text: '邮件发送成功！'
});

$('#demo_icon').msg({
    delay: 0,
    type: 'normal',
    icon: 'icon-reboot',  // 图标名称
    text: '设备重启中，请稍候...'
});

$('#demo_icon').msg({
    delay: 0,
    type: 'error',
    icon: 'icon-lock_fill',  // 图标名称
    text: '验证失败！系统已锁定'
});

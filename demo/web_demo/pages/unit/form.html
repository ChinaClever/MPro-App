表单控件

表单控件用于响应及采集不同类型的用户输入。

基本类型

实例
代码 
<input type="text" class="input" placeholder="文本框">
<input type="password" class="input" placeholder="密码框">
<textarea class="input" rows="3" placeholder="多行文本框"></textarea>
<label>
    <input type="radio" name="radio1" checked>  /* 有 checked 属性，会默认选中 */
    <span>单选1</span>
</label>
<label>
    <input type="radio" name="radio1">
    <span>单选2</span>
</label>
<label>
    <input type="checkbox">
    <span>勾选我</span>
</label>
<label>
    <input type="checkbox">
    <span>还可以勾选我</span>
</label>
<select class="btn">
    <optgroup label="水果">  /* optgroup 用于列表项分组 */
    	<option>苹果</option>
    	<option>香蕉</option>
    	<option>葡萄</option>
    </optgroup>
    <optgroup label="蔬菜">
    	<option>西兰花</option>
    	<option>茄子</option>
    	<option>西红柿</option>
    </optgroup>
</select>
<input type="file" class="input">

不可用状态

1. 为文本框加上 readonly 属性可使其只读（不可写入）。

实例
代码 
<input type="text" class="input" placeholder="文本框只读" readonly>

2. 为表单控件加上 disabled 属性可使其禁用。

实例
代码 
<input type="text" class="input" value="文本框被禁用" disabled>
<textarea class="input" rows="2" disabled>多行文本框被禁用</textarea>
<label>
    <input type="radio" name="radio2" checked disabled>
    <span>单选1</span>
</label>
<label>
    <input type="radio" name="radio2" disabled>
    <span>单选2</span>
</label>
<label>
    <input type="checkbox" disabled>
    <span>勾选我</span>
</label>
<label>
    <input type="checkbox" disabled>
    <span>还可以勾选我</span>
</label>
<select class="btn" disabled>
    ...
</select>
<select class="btn">
    <option>苹果</option>
    <option disabled>香蕉</option>  /* “香蕉”被禁用 */
    <option>葡萄</option>
</select>
<input type="file" class="input" disabled>

表格

表格用于展示列表形式的数据，如日志、报表等。

表格结构

名称
描述
表头
表头一般是表格的第一行，用于描述表格中每一列数据的名称。在垂直显示一组数据的表格中，表头也可以是垂直的第一列。
表头内容一般使用 <th> 标签来包裹。
表体
表体就是表格的内容部分，内容在表体的单元格内，内容可以是文字、图片或其他控件（如文本框、按钮等）。 
<table class="table">
    <thead>
        <tr>
            <th>表头第1列</th>
            <th>表头第2列</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>单元格1行1列</td>
            <td>单元格1行2列</td>
        </tr>
        <tr>
            <td>单元格2行1列</td>
            <td>单元格2行2列</td>
        </tr>
    </tbody>
</table>

隔行交替样式

表格的隔行交替样式能明显提升表格的视觉识别效率，也使得表格更加美观。推荐为内容较长的表格应用该效果，为表格加上 .table-striped 样式即可。

注：IE9 以下的 IE 浏览器无效。

<table class="table table-striped">
...
</table>

响应鼠标悬停

为表格加上 .table-hover 样式，即可实现鼠标悬停时所在行高亮的效果。该效果能形成视觉辅助，方便地查看同一行的内容。

<table class="table table-hover">
...
</table>

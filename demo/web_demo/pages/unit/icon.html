图标

图标能在视觉上简洁有效的指引用户操作。

简介

ARM UI 中的图标采用 iconfont 技术，即图标字体技术：一种将图标集合以字体文件形式进行封装，并可通过 CSS 在 Web 页面上调用的技术。

特点：技术方案成熟，加载文件体积小，矢量无损缩放，样式自定义方便（支持尺寸、颜色等自定义）。

来源：字体资源精选自 阿里巴巴矢量图标库。

使用方法

1. 调用

挑选相应图标获取其类名（图标类名见本文“图标集合”中图标下方 icon-xxx 形式的英文名称），然后应用于页面：

<i class="iconfont icon-xxx"></i>

2. 自定义样式

可以使用 font-size 和 color 样式属性来分别设置图标的尺寸和颜色：

/* 图标大小为40px，颜色为棕色 */
<i class="iconfont icon-arm-ui" style="font-size:40px; color:brown"></i>

3. 场景使用实例

实例
描述

用于按钮，对于一些常用的按钮可以使用图标取代按钮中的文本，使得界面更加简洁 
<button class="btn btn-normal">
    <i class="iconfont icon-collection"></i> 收藏
</button>
<button class="btn btn-primary">
    <i class="iconfont icon-setup_fill"></i>
</button>

资料修改成功！

用于提示消息，图标能指明消息类型，便于用户无需仔细阅读消息内容就可以迅速做出决策 
<div class="msg msg-success">
    <i class="iconfont icon-success_fill"></i>
    <p>资料修改成功！</p>
</div>

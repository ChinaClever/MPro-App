开始使用

新手必看

这部分是给新手看的，主要是对必要的前端知识点进行扫盲，高手请略过。

编写网页项目需要用到以下三种语言：HTML（结构层）、CSS（表现层） 和 JavaScript（行为层）。

HTML 本身并不是编程语言，它用来描述元素应该如何在页面上布局，并向浏览器提供页面所需的其他所有文件列表（例如 CSS 和 JavaScript）。你可以将 HTML 看做盖房的图纸，它可以告诉你房间有多大，里面应该有什么东西。

在编写页面时，你需要使用语义化的 HTML 标签来搭建结构，如果页面布局对你有困难的话，可以参考现成的 模板。

<button>我是按钮</button>  <!-- 普通的 HTML 元素 -->

CSS 负责控制页面的外观。如果说 HTML 描述的是房子的布局，那么 CSS 描述的就是房子的外观。页面上诸如颜色、字体，甚至一些动画都由 CSS 控制。和 HTML 一样，CSS 不是编程语言，它是一种描述型的文本文档，使页面看起来更美观。

使用 ARM UI 编写页面时，你只需为组件根据需要加上对应的 class 类名（组件的样式类名请参见“组件”栏目），即可为当前组件赋以样式风格。例如为按钮添加 btn 和 btn-primary 两个 class 类名，即可为其赋以主要按钮的外观样式。

<button class="btn btn-primary">我是按钮</button>
/* 为按钮加上样式 */
.btn {
    height: 26px;  /* 高度 */
    padding: 0 13px;  /* 内边距 */
    font-size: 12px;  /* 文字大小 */
    color: #fff;  /* 文字颜色 */
    ...
}
.btn-primary {
    background-color: #4680d4;  /* 背景色 */
    border-color: #275ba7;  /* 边框色 */
    ...
}

JavaScript 是编程语言，负责控制页面的交互操作。就像一位勤杂工人，可以拆掉墙壁、建造新的房间和重新装饰房屋。

对于表单组件居多、强调动态交互的后台页面来讲，JavaScript 是非常重要的组成部分。ARM UI 使用 jQuery 库，使我们能够更轻松地进行跨浏览器 DOM 遍历和操作、处理事件和执行 AJAX 操作。

<button class="btn btn-primary" onclick="say()">我是按钮</button>
// 为按钮加上点击事件
function say() {
    alert('你点击了我！');
}

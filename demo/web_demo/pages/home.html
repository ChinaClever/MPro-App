ARM UI 前端框架

一套适用于嵌入式 Web 项目的轻量级前端框架

简介

ARM UI 是一套主要用于嵌入式网页（类似家用路由器的管理网页）项目的前端框架，旨在帮助嵌入式开发工程师快速为其产品搭建一套“看得过去”的网页 GUI 界面，提高开发效率。同样适用于页面交互简单、以设置表单页或报表页为主的后台项目。

定位

ARM UI 面向不熟悉网页开发的工程师服务，解决他们在网页开发上的排版、做图标、写UI样式等方面的痛点。

例如：采用“配置文件+前端工程化工具”的方式来高效构建项目资源，不用人肉去创建文件；使用 iconfont 技术解决图标问题，不再看美工的脸色；编写路由脚本自动实现导航、面包屑和浏览器标题的交互、切换功能等。

特点

  • 轻量级：风格简洁朴素，不作过多修饰，力求每一行代码都精简到极致；
  • 傻瓜化：无需为布局、配色、图标、导航等细节费神，这些经过简单配置后都是自动实现的，另外还提供常用页面模板，可按需复制粘贴；
  • 工程化：采用 NodeJS + Gulp 搭建了工程化实例，可实现编译、压缩、合并、构建等任务，提高开发效率；
  • 可定制：风格、图标均可按需自由定制；
  • 兼容性：兼容主流浏览器，如 Chrome、Firefox 等，兼容IE8 及以上 IE 浏览器。

开始使用

ARM UI 的使用文档最大程度上顾及了前端入门水平的使用者，为他们提供“快速上手”的使用方式；同时也为具有一定前端水平的使用者准备了按需定制、自动化构建等高级使用方式。
